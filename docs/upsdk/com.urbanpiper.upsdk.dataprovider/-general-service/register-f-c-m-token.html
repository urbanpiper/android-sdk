abstract fun registerFCMToken(token: String, deviceId: String, callback: Callback<Void>): CancellableTask

Register the device for FCM

This can be used to force update the application. The response has a field that shows if force update is required. This method should be called when the app is opened and after the user sign's in.

Parameters

  • FCM registration token

deviceId -

  • The unique id of the device

callback -

  • Callback to receive the result

Return
CancellableTask - the request can be cancelled by calling .cancel() on the CancellableTask
