getNearestStore

fun getNearestStore(lat: Double, lng: Double, callback: Callback<StoreResponse>): CancellableTask

Overrides GeneralService.getNearestStore

Returns the nearest store based on lat/ lng

This endpoint helps determine the nearest store from which an order can be delivered. It expects the latitude/longitude information of the location to be delivered to. Along with the store information, this endpoint also returns the biz related information, which can be cached for later use.

Parameters

lat -

  • User's latitude

lng -

  • User's longitude

callback -

  • Callback to receive the result

Return
CancellableTask - the request can be cancelled by calling .cancel() on the CancellableTask
