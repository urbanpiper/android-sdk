validateCart

fun validateCart(order: Order, callback: Callback<ValidateCartResponse>): CancellableTask

Validate cart should be called every time the cart changes, if user is in the cart page. This method pre-processes the contents of the cart to return the computational details for the order, like the charges, taxes, total, etc. We strongly recommend client applications not to perform these complex computations at their end, since there are many variables that can affect the computations—not all of which are available with the client application at any time.
