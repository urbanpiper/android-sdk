abstract fun getFilteredItems(categoryId: Int, locationId: String, filterBy: String, offset: Int, limit: Int, callback: Callback<CategoryItemResponse>): CancellableTask

returns a list of active items filtered based on the filters passed

This method returns a list of active items filtered based on filters received from the getFilterAndSortOptions method for a category from which an order can be placed. Multiple filters can be sent as comma separated values (eg - filter1,filter2,filter3)
